# recosphere-recommendation-engine
RecoSphere is a recommendation system that suggests products based on similarity using machine learning techniques.

## Features
- Item-based recommendations  
- Similarity calculation  
- Interactive UI  

## Tech Stack
- Python  
- Scikit-learn  
- Streamlit  

## Run
streamlit run app.py

## How It Works
- Computes similarity between items  
- Recommends most similar items  

## Future Improvements
- User-based recommendations  
- Deep learning models
